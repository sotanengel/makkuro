# Contributor Covenant Code of Conduct

## 私たちの約束

私たち、メンバー、コントリビュータ、リーダは、年齢、体型、目に見える
あるいは見えない障害、民族、性別、ジェンダーアイデンティティと表現、
経験レベル、教育、社会経済的地位、国籍、外見、人種、宗教、性的指向、
性的同一性に関係なく、ハラスメントのないコミュニティとするよう努めます。

私たちは、開かれた、歓迎的な、多様で、包括的で、健全なコミュニティに
貢献する形で行動し、交流します。

## 私たちの基準

コミュニティにポジティブな環境を作る行動の例:

- 他者への共感と優しさを示す
- 異なる意見、視点、経験を尊重する
- 建設的なフィードバックを与え、優雅に受け入れる
- 自らの過ちの影響を認めて責任を持ち謝罪し、その経験から学ぶ
- 個人としてだけでなく、コミュニティ全体にとって最善となることに
  焦点を当てる

容認できない行動の例:

- 性的な言葉や画像の使用、あらゆる種類の性的注目または誘い
- トローリング、侮辱的または軽蔑的なコメント、人身攻撃または政治的攻撃
- 公の場または非公開の場でのハラスメント
- 物理的な住所やメールアドレスなど、他者の私的情報を明示的な許可なく
  公開すること
- 職業上不適切と合理的にみなされるその他の行為

## 実施責任

コミュニティリーダは、容認できる行動基準を明確にし、実施する責任を負い、
不適切、脅迫的、攻撃的、または有害とみなされる行動に対して適切かつ公正な
是正措置を取ります。

## 適用範囲

この行動規範は、コミュニティ空間全体に適用されます。また、公的な場で
コミュニティを代表している場合にも適用されます。

## 執行

虐待、ハラスメント、またはその他容認できない行動の事例は、GitHub Security
Advisory ないし repo オーナーへの個別連絡で報告してください。すべての
苦情は速やかかつ公正に審査・調査されます。

コミュニティリーダは、この行動規範に違反したとみなされた者の報告者の
プライバシーと安全を尊重する義務があります。
